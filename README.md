# CodSoft Internship — AI & Python Projects

> AI and Python projects completed during the CodSoft Virtual Internship Program.

---

## About

This repository contains tasks and projects completed as part of the **CodSoft Virtual Internship**. The work covers AI fundamentals, recommendation systems, and game AI — all implemented in Python.

---

## Projects

### 🤖 AI Chatbot
A rule-based conversational chatbot built in Python that responds to user inputs using pattern matching and predefined responses.

**Tech:** Python, NLP concepts, if-else / pattern matching logic  
**Concepts covered:** Conversational flow, input parsing, response generation

---

### 🎬 Recommendation System
A content-based recommendation system that suggests items to users based on their preferences and similarity scores.

**Tech:** Python, Pandas, Scikit-learn (cosine similarity / TF-IDF)  
**Concepts covered:** Content-based filtering, similarity computation, data preprocessing

---

### ❌⭕ Tic Tac Toe (AI vs Human)
An interactive Tic Tac Toe game where a human plays against an AI opponent implemented using the **Minimax algorithm**.

**Tech:** Python, Minimax algorithm, game logic  
**Concepts covered:** Game trees, recursive search, optimal decision making
